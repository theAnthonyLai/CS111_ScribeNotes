Grading and Distribution

  • Exams (1/3)
    • 1 midterm (during lecture), 100 minutes, (1/9)
    • 1 final, 180 minutes, (2/9)
  • Labs (1/3)

    • There are 4 labs which worth 1/12 each. Labs can be done in team of 2

    Breakdown topics (tentatively)

    • Shell
    • Kernel Hacking
    • File System
    • Networking
  • Minilabs (2/15)

    • There are 2 minilabs which worth 1/15 each. Minitlabs are to be done in individuals

    Breakdown topics (tentatively)

    • Scheduling
    • Virtual Memory
  • Lab Extensions (1/12)

    • There will be 1 design problem, which contains written report, to be done in teams of 2.

  • Paper (1/15)

    • There is one 2-4 pages paper on a contemporary operating system topic

  • Scribe Notes (1/20)

    • Scribe notes are to be done in groups of up to 4 students. They are due one week after the lecture.
